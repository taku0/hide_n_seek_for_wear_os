# Hide 'n' seek watch face for Wear OS

High contrast and high data-ink ratio analog clock for Wear OS.

## Features

- High legibility.
- Optimized for ambient (always-on) mode.
- Five complications.
- Vibrating once (or customizable times) every hour.
